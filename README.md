# ✦ devlog — GitHub Pages 블로그 템플릿

단일 HTML 파일로 동작하는 미니멀 블로그 템플릿입니다.  
GitHub Pages에 올리면 즉시 사용 가능합니다.

---

## 🚀 GitHub Pages 배포 방법

1. 이 저장소를 Fork하거나, `index.html`을 자신의 저장소에 복사
2. 저장소 이름을 `username.github.io`로 설정 (개인 페이지) 또는 아무 이름으로 설정 (프로젝트 페이지)
3. **Settings → Pages → Source** 에서 브랜치를 `main`으로 설정
4. 잠시 후 `https://username.github.io` 에서 확인

---

## ✏️ 글 추가 방법

### 방법 1: 브라우저에서 직접 작성 (로컬/임시)
- 상단 **[+ 글쓰기]** 버튼 → 작성 → 저장  
- ⚠️ 브라우저 localStorage에 저장되므로, 다른 기기에서는 보이지 않음

### 방법 2: `index.html` 파일 직접 편집 (영구 저장 권장)
`index.html` 파일 하단 `<script>` 부분의 `posts` 배열에 글을 추가합니다:

```javascript
const posts = [
  {
    id: 1,                          // 고유 숫자 ID
    title: '글 제목',
    category: '개발',               // 개발 / 일상 / 독서 / 여행 / 기타
    tags: ['태그1', '태그2'],
    summary: '목록에 보일 짧은 설명',
    body: `# 제목\n\n본문 내용...`, // 마크다운 사용 가능
    date: '2025-01-15',             // YYYY-MM-DD 형식
  },
  // 글 추가...
];
```

수정 후 `git push`하면 GitHub Pages에 자동 반영됩니다.

|------|------|
| 사이드바 토글 | 좌상단 햄버거 버튼으로 접기/펼치기 |

## 🎨 커스터마이징

### 블로그 제목 변경
```html
<!-- nav 영역에서 수정 -->
<span class="nav-logo">✦ devlog</span>

<!-- 브라우저 탭 제목 -->
<title>My Blog</title>
```

### 카테고리 추가
1. `index.html` 사이드바 HTML에 카테고리 버튼 추가
2. `CAT_COLORS` 객체에 색상 추가
3. 글쓰기 폼의 `<select>` 에도 `<option>` 추가

### 색상 테마 변경
`<style>` 내 `:root` 의 CSS 변수를 수정하세요:
```css
:root {
  --accent: #c8ff00;   /* 강조 색상 */
  --bg: #0d0d0f;       /* 배경색 */
}
```
